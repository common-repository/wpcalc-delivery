Calculator for the carriage of goods from one city to another

== Description ==

Wpcalc Delivery plugin to calculate the cost of transportation of goods. This plug is suitable for transport companies that carry freight between cities. 
Easy integration calculator on the page or post by using the shortcode.

For proper operation of the plugin, you will need to install a plugin Wpcalc Modal Form

== Installation ==

1. Upload `wpcalc-delivery` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings (page Wpcalc Delivery).

To insert a contact form on the page or post website use shortcode [wpcdelivery]
